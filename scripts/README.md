# ⚒️ scripts
**The Maintenance Tunnel**

Rust and Shell utilities for forge upkeep. All scripts are intended to be run from the workspace root.

### Key Scripts:
- **`fix_ci_issues.sh`**: Quick-fix for common CI failures.
- **`setup_cross_compile.sh`**: Environment prep for aarch64 builds.
- **`document_hygiene.rs`**: (Binary) Automated archival of old scratch docs.
- **`auto_bump.rs`**: (Binary) Workspace-wide version management.
