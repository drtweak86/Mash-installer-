# 🎁 pkg
**The Export Dock**

Packaging definitions and templates for distributing MASH across various realms.

### Contents:
- **`aur/`**: Arch User Repository PKGBUILD and .SRCINFO.
- **`deb/`**: Debian package templates (managed via cargo-deb).
- **`rpm/`**: RPM package templates (managed via cargo-generate-rpm).
