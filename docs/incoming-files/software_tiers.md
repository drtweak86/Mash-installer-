# Goblin-Tiered Software List

This ledger page records the curated S/A-tier canon that the mash installer now offers through the interactive software tier menu. Every category is mentioned so the bard can point the miners toward the best blades and the alternatives, and the CLI gives the dwarves two moods: a **full install** that picks the S-tier champion or a **custom run** that lets them choose any of the five entries shown for that category.

Each category below lists the recommended S-tier champion first, followed by four A-tier alternatives.

## Terminal
- **S-tier:** Kitty – GPU-accelerated, ligature-ready, terminal chroma with scrollback and tabs.
- **A-tier:** Alacritty, WezTerm, Foot, ST.

## Shell
- **S-tier:** Zsh + Starship – the mash-approved combo that layers the Starship prompt on Zsh.
- **A-tier:** Fish, Bash, Nu, PowerShell Core.

## File Manager
- **S-tier:** eza – iconic `ls` upgrade with icons, Git awareness, and tree views built in.
- **A-tier:** lf, nnn, ranger, vifm.

## Text Editor
- **S-tier:** Helix – Rust-based, modal, and ready for modern workflows.
- **A-tier:** Neovim, Visual Studio Code, Micro, Kakoune.

## Git Client
- **S-tier:** Lazygit – TUI staging, rebasing, and history explorer in one.
- **A-tier:** Tig, GitUI, Forge, GitHub CLI (`gh`).

## Process Viewer
- **S-tier:** btop – retina-safe stats, graphs, and widgets.
- **A-tier:** glances, htop, bpytop, gotop.

## Browser
- **S-tier:** Brave – syncable, privacy-first Chromium with built-in shields.
- **A-tier:** Librewolf, Vivaldi, Firefox, Chromium.

## Media Player
- **S-tier:** MPV – scriptable, lightweight, and codec-agnostic.
- **A-tier:** VLC, SMPlayer, Celluloid, MPlayer.

## HTPC
- **S-tier:** Kodi – remote-ready home theater with skins and remotes.
- **A-tier:** Plex Media Server, Jellyfin, Emby, OSMC.

## VPN
- **S-tier:** WireGuard – lean configs, modern crypto, best-in-class performance.
- **A-tier:** OpenVPN, Tailscale, StrongSwan, OpenConnect.

## Firewall
- **S-tier:** nftables – the modern packet filter that replaces iptables.
- **A-tier:** firewalld, ufw, iptables, Shorewall.

## Backup
- **S-tier:** rclone + borg – sync clouds with rclone, snapshot with borg.
- **A-tier:** restic, duplicacy, duplicity, borgmatic.
